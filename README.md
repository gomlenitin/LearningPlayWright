# LearningPlayWright

This repository contains code and examples for learning Playwright using JavaScript.

## Setup

1.  Clone the repository:
    ```bash
    git clone https://github.com/gomlenitin/LearningPlayWright.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

## Structure

-   `Chapter_01_Basics`: Basic Playwright concepts and setup.
-   `Chapter_02_Java_Concept`: Core JavaScript concepts including Variables, Data Types, Hoisting, Scope (let, var, const), and functions.

## Running Tests

To run the tests, use the standard Playwright commands:

```bash
npx playwright test
```
